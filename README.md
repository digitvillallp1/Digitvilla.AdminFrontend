# Digitvilla Admin Frontend

A modern, responsive web application for managing online fee payments. Built with a focus on performance, scalability, and clean user experience, this application provides essential tools for payment tracking, administration, and reporting.

---

## Overview

This project is designed to simulate and manage fee-related workflows through a user-friendly interface. It includes dashboards, payment flows, and administrative controls suitable for internal management systems or demonstration purposes.

---

## Features

* User authentication (simulated)
* Dashboard with fee summaries and status tracking
* Month-based payment selection
* Payment processing simulation
* Receipt generation
* Administrative dashboard
* Responsive layout for desktop and mobile devices

---

## Technology Stack

* React 18 with Vite
* TypeScript
* Tailwind CSS
* shadcn/ui component library
* React Router
* Zustand for state management
* React Hook Form with Zod for validation
* Recharts for data visualization

---

## Getting Started

### Prerequisites

Ensure you have the following installed:

* Node.js (v16 or higher recommended)
* npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/digitvillallp1/Digitvilla.AdminFrontend.git
cd Digitvilla.AdminFrontend
```

Install dependencies:

```bash
npm install
```

### Running the Application

Start the development server:

```bash
npm run dev
```

The application will be available at the local development URL provided in the terminal.

## Project Structure

```
src/
  components/     Reusable UI components
  pages/          Application pages
  store/          State management (Zustand)
  hooks/          Custom React hooks
  utils/          Utility functions
  assets/         Static assets
```

## Build and Deployment

To create a production build:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```
